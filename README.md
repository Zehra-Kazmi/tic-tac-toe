## 🎮 How to Play
1. The game is played on a **3x3 grid**.
2. **Player X** always goes first, followed by **Player O**.
3. Players take turns clicking on empty cells to place their respective marks (`X` or `O`).
4. The **first player to get 3 of their marks in a row** (up, down, across, or diagonally) is declared the winner!
5. If all 9 cells are filled and no player has 3 marks in a row, the game ends in a **Draw**.
6. Click the **"Play Again"** button at any time to clear the board and start a brand new game.

## ✨ Features
* **Modern UI:** Built with a clean cyber-punk color palette and smooth CSS transitions.
* **Responsive Design:** Optimized to work perfectly on mobile screens, tablets, and desktop monitors.
* **Winner Highlights:** Automatically checks for winning patterns and highlights the winning 3-in-a-row cells in green.
* **Instant Reset:** Allows quick board resets without needing to reload the entire web page.
